# Bayesian Learning SGLD

A small Python project demonstrating Bayesian learning with Stochastic Gradient Langevin Dynamics (SGLD).

## Contents

- `src/` — Python modules and experiment scripts
- `notebooks/` — Jupyter notebooks for logistic regression and demo exploration
- `requirements.txt` — project dependencies

## Setup

1. Create a virtual environment:
   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

## Run

- `python src/experiments/logistic_regression.py`
- `python src/experiments/simple_demo.py`
- `python src/experiments/ica.py`

## Notebooks
- `simple_demo.ipynb`
- `logistic_regression_sgld.ipynb`
- `ica_sgld.ipynb`
