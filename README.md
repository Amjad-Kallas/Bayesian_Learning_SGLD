# Bayesian Learning SGLD

A small Python project demonstrating Bayesian learning with Stochastic Gradient Langevin Dynamics (SGLD).

## Contents

- `src/` — Python modules and experiment scripts
- `notebooks/` — Jupyter notebooks for all experiments
- `requirements.txt` — project dependencies

## Setup

1. Create a virtual environment:
   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

## Run

- `python -m src.experiments.logistic_regression`
- `python -m src.experiments.simple_demo`
- `python -m src.experiments.ica`

## Notebooks
- `simple_demo.ipynb`
- `logistic_regression_sgld.ipynb`
- `ica_sgld.ipynb`
